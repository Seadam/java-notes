### 问题

* 首页显示不正常，没有加载 banner 轮播图
  * 缺 JS／CSS 文件，复制进去就行
* JSP 回到主页按钮的路径要输入正确端口号
  * `${pageContext.request.contextPath }/index.jsp`
* 没有登录进入个人中心页面显示错误，应该跳转回首页
  * 如果用户没有登录，跳转回首页，**Filter** 判断
* 数据库模糊查询，输入关键字时
  * `% + param + %`
